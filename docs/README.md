# Queenshift Public Docs

This folder holds the main supporting docs for the public Queenshift repo.

Current public-doc entry points:

1. `../README.md` for the product overview
2. `../QUICKSTART.md` for the calmest first run
3. `../AUTHORS.md`, `../CITATION.cff`, `../.zenodo.json`, `../CHANGELOG.md`, `../CONTRIBUTING.md`, `../CODE_OF_CONDUCT.md`, `../SECURITY.md`, and `../LICENSE` for the release and governance surface
4. `install.md` for install truth and source-checkout boundaries
5. `providers.md` for provider setup and fail-closed rules
6. `task-families.md` for calmer first tasks versus bounded follow-on families
7. `evidence.md` for one stranger-readable summary of the current live anchors, experimental boundaries, and unresolved benchmark picture
8. `../.github/ISSUE_TEMPLATE/` plus `../.github/pull_request_template.md` for public bug, task-family, and PR intake

These docs stay stranger-first and bounded. Private maintainer planning docs, launch drafts, and export-prep notes are intentionally kept out of this public repo.
