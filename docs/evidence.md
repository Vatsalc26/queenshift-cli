# Queenshift Evidence And Benchmark Reading

This page is the public evidence summary for Queenshift as of `2026-03-30`.

Use it when you want one truthful answer for what is live now, what is still experimental, and what the project does not yet claim.

## What Is Live Now

1. `Queenshift` is a bounded coding CLI for small clean Git repos.
2. `swarmengine` remains the shipped bounded engine.
3. the public command surface is `queenshift`.
4. the public first-run path now teaches `queenshift doctor`, the guided demo lane, the disposable demo lane, `repo:onboard`, `--admitOnly`, and the bounded recovery commands.

## What Stays Experimental

1. `queenbee` remains experimental and is not the shipped default engine.
2. as of `2026-03-29`, the current live QueenBee evidence has two provider-backed anchors only:
   `QB-LIVE-01` as one canonical live row and `QB-LIVE-GW-01` as one gateway live row.
3. the remaining canonical, natural-language, and gateway QueenBee rows still stay on explicit hold or proof-backed-only status.
4. this evidence does not justify a default-engine change, a broad public QueenBee claim, or a cross-tool victory claim.

## Benchmark Reading

1. the fixed benchmark pack now covers all `10` current task families once.
2. the same-day cross-tool study is still incomplete.
3. the current recorded scoreboard remains wins=`0`, ties=`0`, trails=`0`, unresolved=`30/30`.
4. `not_run` is not a tie and is not a hidden win.
5. the current benchmark story is therefore an honest gap review, not a leaderboard claim.

## What Queenshift Can Claim

1. a bounded coding CLI with artifact-backed review, replay, and incident follow-up.
2. a calmer first-run path with productized `queenshift` commands.
3. one shipped bounded engine and one clearly separated experimental engine candidate.
4. public docs that now match the real CLI instead of older early-stage command habits.

## What Queenshift Does Not Claim Yet

1. broad general-use autonomous coding.
2. open-ended repo-wide refactors.
3. packaged cross-platform parity beyond the current bounded install story.
4. `queenbee` as the shipped public engine.
5. a measured cross-tool benchmark win.

## Short Launch Summary

Use this launch-safe summary:

Queenshift is a bounded coding CLI for small clean Git repos. `swarmengine` remains the shipped bounded engine, `queenbee` remains experimental, `QB-LIVE-01` plus `QB-LIVE-GW-01` are the current provider-backed live anchors, and the fixed cross-tool benchmark scoreboard is still unresolved rather than promotional.
