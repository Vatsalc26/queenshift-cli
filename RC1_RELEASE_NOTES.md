# RC1 Release Notes

## Version

`0.1.0-rc1`

## What This Release Represents

This is the first serious public code release for Queenshift.

It should be read as:

1. a bounded source release
2. a governance-first engineering CLI slice
3. a public software counterpart to the v0.1 technical note

It should not be read as:

1. broad general-use autonomy
2. full parity with top-tier coding CLIs
3. proof that every repo shape is now supported

## Current Public Truth

1. `swarmengine` remains the shipped bounded engine
2. `queenbee` remains experimental
3. the stricter normal-user production-ready answer is still `NO`
4. the public repo is real and useful, but still bounded
